#Atrapacorazones
Es un juego en donde tienes que atrapar la mayor cantidad de corazones que caen evitando que se te escape uno, tienes solo **3 oportunidades**. 

##Controles. 
Te desplazas con [A] por la izquierda, [D] por la derecha y [Q] para salir del juego. 

##Compilacion y Ejecucion. 

'''
git clone https://github.com/nerdemma/atrapacorazones
cd atrapacorazones
make main
./main
'''
